SIGNFLOW - SIGN LANGUAGE TRANSLATOR
====================================

SETUP INSTRUCTIONS:

1. Backend Setup:
   - Install Python 3.7+
   - Run: pip install -r requirements.txt
   - Run: python backend.py
   - Backend will start on http://localhost:5000

2. Frontend Setup:
   - Simply open signflow.html in a web browser
   - Allow camera access when prompted

FEATURES:
- Real-time image capture every 0.5 seconds
- Webcam integration
- Text-to-speech conversion
- Beautiful, responsive UI
- Image processing backend
